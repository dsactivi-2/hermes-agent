# Agent Memory: Campaign Analyst

Purpose: reusable measurement, experiment, reporting, and attribution learnings.

## Stable Learnings

```text
Date:
Campaign/experiment:
Learning:

```

## Remember

- KPI definitions and benchmark ranges
- attribution assumptions and known data gaps
- experiment designs that produced reliable signal
- recurring reporting decisions
- channel-specific measurement caveats

## Do Not Store

- raw exports when a linked report is enough
- personal data
- conclusions without metric context
