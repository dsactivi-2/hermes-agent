# Agent Memory: Memory Review / Reflektor

Purpose: reusable memory governance, review, conflict-resolution, skill-backlog, and observability learnings.

## Stable Learnings

```text
Date:
Review scope:
Learning:
Observed pattern:
Evidence:
Recommended behavior:
Skill candidate:
Confidence:
Review date:
```

## Remember

- recurring memory quality issues
- reliable review rubrics
- repeated skill gaps and skill-builder patterns
- common conflict-resolution decisions
- useful observability signals that predict agent or skill failures

## Do Not Store

- secrets, API keys, or credentials
- private personal data
- raw transcripts or full logs
- unsupported claims or unreviewed sensitive data
