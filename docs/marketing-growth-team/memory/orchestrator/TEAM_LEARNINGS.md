## Promotion Rules

- Promote learnings from agent memory when they affect more than one agent.
- If a learning repeats at least twice and has a clear procedure, add it to `SKILL_BACKLOG.md`.
- If a learning is only relevant to one role, keep it in that agent's memory file.
