## Source Quality Rubric

- Prefer primary sources, official documentation, customer-owned data, and current platform docs.
- Use recent sources for platform rules, pricing, benchmarks, trends, and legal/compliance-sensitive topics.
- Mark inference clearly when a conclusion is not directly stated by a source.
