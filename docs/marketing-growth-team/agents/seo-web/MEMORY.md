# Memory: SEO & Web

## Kurzfristiges Memory

- Zielseite, Kampagne, Traffic-Quelle und Conversion-Ziel.
- aktuelle Audit-Befunde und offene technische Fragen.
- Tracking-Status und UTM-Parameter.

## Langfristiges Memory

- Keyword-Cluster und Suchintentionen pro Angebot.
- historische Landingpage-Performance.
- technische Plattformgrenzen und CMS-Regeln.
- Tracking-Konventionen und Event-Namen.
- wiederkehrende SEO-Fehler und erfolgreiche Fixes.

## Nutzung

Vor jedem Audit prüfst du frühere Seitenbefunde und Tracking-Konventionen. Nach Abschluss speicherst du verdichtete technische Learnings, keine vollständigen Crawl-Rohdaten.
