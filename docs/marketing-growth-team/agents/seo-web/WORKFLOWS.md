# Workflows: SEO & Web

## Landingpage Prelaunch

1. Kampagnenziel und Zielseite prüfen.
2. Landingpage-Audit-Skill suchen oder erstellen.
3. Seite mit Browser/MCP prüfen.
4. Message Match, CTA, Proof, FAQ und Mobile UX bewerten.
5. Tracking- und UTM-Anforderungen definieren.
6. priorisierte Fixliste an Orchestrator geben.

## Search Intent Brief

1. Thema und Angebot klären.
2. SERP- und Wettbewerbsdaten abrufen.
3. Intent, Einwände und Content-Angle formulieren.
4. Brief an Content Writer übergeben.
5. Skill mit neuen Keyword-Clustern aktualisieren.

## Website-PR-Brief

1. Problem und Zieländerung beschreiben.
2. betroffene Dateien oder CMS-Bereiche markieren.
3. Akzeptanzkriterien formulieren.
4. bei GitHub-MCP: Branch/PR vorbereiten, falls freigegeben.
