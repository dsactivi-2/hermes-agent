# Skills: SEO & Web

## Bereits sinnvoll nutzbare Skills

- `creative/popular-web-designs`: moderne Landingpage-Patterns und UX-Strukturen.
- `web-development/page-agent`: Seitenanalyse und webnahe Umsetzung.
- `research/duckduckgo-search` oder `research/searxng-search`: Recherche bei fehlendem Web-Plugin.

## Autonom zu erstellende Skills

- `marketing-growth/landing-page-audit`: Message Match, UX, Conversion, SEO, Tracking.
- `marketing-growth/utm-taxonomy`: konsistente UTM-Namensgebung und Validierung.
- `marketing-growth/search-intent-brief`: Keyword, SERP, Intent, Content-Angle.
- `marketing-growth/prelaunch-tracking-check`: Events, Ziele, Consent, Testkonversion.
- `marketing-growth/seo-content-refresh`: bestehende Seiten gegen neue Kampagnenziele prüfen.

## Erstellungskriterien

Erstelle eine Skill, wenn ein Audit mehrere Kampagnen betrifft, Tracking-Regeln wiederholt gebraucht werden, ein Check mehr als fünf Prüfpunkte umfasst oder eine Website-Änderung regelmäßig als PR/Ticket vorbereitet wird.
