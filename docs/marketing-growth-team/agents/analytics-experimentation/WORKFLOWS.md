# Workflows

## Experiment Spec

1. Define hypothesis, audience, intervention, primary metric, guardrails, and duration.
2. Define assignment, holdout, instrumentation, and decision rule.
3. Identify risks and approval requirements.

## Metrics Tree

1. Start from business objective.
2. Map input, output, quality, and guardrail metrics.
3. Define source of truth and owner for each metric.

## Readout

1. Pull approved data.
2. Compare against hypothesis and guardrails.
3. Document caveats, decision, and follow-up.
