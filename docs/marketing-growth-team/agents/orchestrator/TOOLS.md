# Tools

Preferred access order:

1. Existing Hermes skills for planning, research, writing, and project work.
2. MCP servers configured for search, docs, analytics, CRM, and task tracking.
3. Repository scripts or CLI commands.
4. Plugins or service-gated tools.
5. New core tools only as a last resort and only after documenting why the
   Footprint Ladder cannot satisfy the need.

Expected integrations:

- Search MCP for current market evidence.
- Docs MCP for brand, product, and internal planning docs.
- Project management MCP for task creation after approval.
- Analytics MCP for metric validation.
