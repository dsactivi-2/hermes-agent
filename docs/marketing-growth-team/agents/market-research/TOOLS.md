# Tools

Preferred tools:

- Search MCP for public web research.
- Docs MCP for internal brand, sales, and customer notes.
- Analytics MCP for demand, traffic, and cohort data.
- Spreadsheet or warehouse MCP for survey and CRM exports.
- Existing research skills for structured summaries.

Avoid:

- New core tools for one-off research tasks.
- Unapproved scraping of restricted or personal data.
